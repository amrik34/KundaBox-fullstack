# Task Solution

## Backend Setup

1. Clone repo
2. cd backend
3. npm install
4. npm start
   - Runs Express server with insert_user API
5. npm test
   - Runs unit tests

## Frontend Setup

1. cd frontend
2. npm install
3. npm start
   - Runs React app (or vanilla HTML/JS if kept simple)

## Usage

- Open http://localhost:3000 for frontend.
- API runs at http://localhost:5000 (for user insert tests).
